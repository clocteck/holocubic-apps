硬件监控面板

AIDA Monitor

连接 AIDA64 RemoteSensor，在设备上显示 CPU、GPU、内存和温度数据。

AIDA64 配置

  1. 1 开启 RemoteSensor在 AIDA64 中进入 File > Preferences > Hardware Monitoring > LCD，启用 RemoteSensor / LCD 支持。
  2. 2 导入 LCD 项目打开控制页下载 holo-aida.rslcd，然后在 AIDA64 的 设置 > LCD > LCD 项目 > 导入 中导入。
  3. 3 确认访问地址电脑和设备需在同一网络，浏览器访问 RemoteSensor 地址应能看到 data: 开头的数据。http://<pc-ip>:80/sse
  4. 4 保存电脑 IP进入应用控制页，填写 AIDA64 所在电脑的 IP；如果 AIDA64 使用了其他端口，也同步修改 port。config.host = "192.168.0.80"
    config.port = 80
    config.path = "/sse"
CPU Usage CPU Frequency CPU Temperature GPU Usage GPU Frequency GPU Temperature Memory Usage

如果页面只返回 data: ReLoad，说明 RemoteSensor 已启动但还没有输出项目；请在 LCD 项目中导入配置并点击 Apply。
